# BA_Furniture / Noi That Phu Vuong - Floor Zoning Plan

## Nguyen tac doc so do

- FRONT = mat tien / duong / khu khach nhin thay dau tien.
- BACK = phia sau nha.
- CORE = thang may/thang hang, thang bo, WC, truc ky thuat.
- Customer route = tuyen khach.
- Goods route = tuyen hang.

So do duoi day la zoning cong nang, khong phai ban ve thi cong. Ket cau se doi chieu sau.

---

## Tang 1 - Kho nhap xuat / logistics

```text
FRONT / STREET
┌──────────────────────────────────────────┐
│  CUA CUON / NHAP HANG   │ LOI NGUOI     │
│  khu xe dung ngan han    │ vao rieng     │
├──────────────────────────┴───────────────┤
│  TAM DUNG HANG / KIEM HANG / DAN MA      │
├──────────────────────────────────────────┤
│  KHO CHINH + KE HANG + LOI DI HANG       │
│                                          │
│  Goods route -> -> -> CORE / THANG HANG  │
├──────────────────────────────────────────┤
│  CORE: thang hang/thang may + thang bo   │
│  WC / ky thuat / kho phu                 │
└──────────────────────────────────────────┘
BACK
```

### Ghi chu
- Tang 1 uu tien kho va van hanh.
- Loi khach neu co chi nen la loi ro rang, sach, ngan, khong cat ngang khu hang lon.
- Thang hang can gan tuyen kho.
- Cua cuon mat tien phai du rong de nhap/xuat hang.

### Customer route
Mat tien -> loi nguoi -> core/thang -> len tang 2/3/4.

### Goods route
Mat tien -> cua cuon -> tam dung -> kiem hang -> kho -> thang hang.

## Tong ket tuyen dung

```text
CUSTOMER:
Street -> Entrance -> Core -> T2 Consult -> T3 Office Showroom -> T4 Home Showroom -> Consult -> Exit

GOODS:
Street -> Roller Door -> Receiving -> Check/Label -> Warehouse -> Goods Lift -> Display/Floor Stock

STAFF:
Entrance/Core -> T2 Office -> T1 Warehouse / T3-T4 Showroom / T5 Studio as needed
```

## Ranh gioi can giu

- Tang 1: logistics first.
- Tang 2: office and consulting first.
- Tang 3: office furniture sales first.
- Tang 4: home/living experience first.
- Tang 5: content/support first.
